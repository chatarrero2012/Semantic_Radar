# 🧠 Semantic Radar

> Map the invisible. Understand the noise. Navigate the narrative.

Semantic Radar is a lightweight, offline-first tool that transforms raw text streams into interactive semantic maps.  
Designed for creators, analysts, and curious minds who want to **see patterns instead of drowning in them**.

---

## 🌌 What it does

- 🧭 Maps text into a **3D semantic space**
- 🔍 Reveals hidden structure in conversations
- ⚡ Works in real-time with minimal resources
- 🧯 Survives unstable environments (VMs, old machines, chaos)

---

## 🎯 Why it exists

Digital conversations are noisy, emotional, and often manipulated.

Semantic Radar doesn't try to "judge" content.  
It helps you **see the structure behind it**.

> Not opinion. Not moderation.  
> Just signal emerging from noise.

---

## ⚙️ Features

- 🟢 **Safe Mode (ASCII vector)**
  - Ultra-stable
  - Runs anywhere
  - Zero dependencies drama

- 🔵 **Embedding Mode**
  - Real semantic understanding
  - Uses `sentence-transformers`
  - Auto fallback if something breaks

- 🌌 **3D Visualization**
  - Interactive semantic trajectory
  - Pattern-aware, not just pretty

- 🧠 **Resilient Architecture**
  - Thread-safe
  - Crash-resistant
  - Designed for imperfect environments
